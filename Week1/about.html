About the Developer

This site is a collection of the games that were created in the HTML/JavaScript Class Spring 2026 at New England Institute of Technology. The class used HTML, CSS, JavaScript, and the HTML Canvas to learn and create video games.

Each game is built from scratch using core technologies with a focus on foundational design and performance for the web.

More Games to be featured soon, Jordan Dubreuil is a Professor in Video Game Design at NEIT and here are example projects for his students in class.
